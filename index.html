我是周蕊蝶，因朋友了解到UI/UX的工作領域，產生濃厚的興趣並開始接觸 HTML 與 CSS。從對語法一竅不通，到現在能創作出各種有生命的網頁，勇於挑戰、實踐、有耐心不怕嘗試是我的代名詞。希望能創造出更多有趣、有美感兼具實用性的介面，成為專業的UI/UX前端開發人員。

SKILL

HTML

對網頁效能有概念，有能力看著設計稿做出文字的編排、架構，以及設計對使用者友善的體驗，有系統的規劃整體內容。

CSS
